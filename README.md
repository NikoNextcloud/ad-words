# TrendCraft BG

Безплатно статично приложение за:

- подготовка на Google Trends проучване за България;
- импорт и анализ на CSV данни;
- подреждане на ключови фрази;
- създаване на текстове за Facebook и Instagram.

## Публикуване в GitHub Pages

1. Качете всички файлове от тази папка в основната папка на ново публично GitHub хранилище.
2. Отворете **Settings → Pages**.
3. Под **Build and deployment** изберете **Deploy from a branch**.
4. Изберете клона **main**, папка **/(root)** и натиснете **Save**.

Приложението не използва платени API услуги. Всички въведени данни и генерирани текстове се пазят само в браузъра чрез `localStorage`.

Google Trends показва относителен интерес, не точен брой месечни търсения.
